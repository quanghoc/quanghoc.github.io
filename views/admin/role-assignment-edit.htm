Tenant Roles Roles which determine what functions Users/Groups can perform within the Tenant.

Provider Tenant

Grant Role Description
Tenant Administrator
  • Creates projects and has all permissions for projects in their tenant.
  • Sets ACL permissions for projects and the service catalog.
  • Assigns tenant roles.
  • Manages resources for all projects in their tenant.
  • Creates and manages subtenants.
Project Administrator
  • Creates projects in their tenant.
Tenant Approver
  • Approves orders in their tenant.
Virtual Data Center Roles Roles which determine what functions Users/Groups can perform within the Virtual Data Center.
Grant Role Description
System Administrator
  • Manages the Virtual Data Center physical and virtual assets.
  • Manages software and license updates.
  • Retrieves ViPR status and system health.
  • Configures Data Services.
Security Administrator
  • Manages the Virtual Data Center authentication providers.
  • Sets system configuration parameters for the Virtual Data Center.
  • Assigns roles for the Virtual Data Center and for tenants.
  • Manage Virtual Data Centers in a GEO Configuration.
System Monitor
  • Retrieves metering and monitoring records for the Virtual Data Center.
System Auditor
  • Retrieves audit logs for the Virtual Data Center.
