Data Services Nodes

The following steps will guide you through the process to deploy the ViPR Data Services virtual machines within your environment. Please note that you need to complete the steps below before you create your first Data Store.


  • Configure

    Under Configuration Properties, update the Extra Node IP addressesproperty to include the IP addresses or subnets of the data services nodes you are adding. This will require a reboot of the ViPR appliance. This will allow the data services nodes to connect to the ViPR appliance.

  • Download

    Download the Data Services Configuration File. Ensure the configuration file name is config.iso. Copy the config.isofile to the same directory where the Data Service deployment files were extracted (OVF/VMDK).


    Download Configuration File

  • Deploy

    Deploy the dataservice.ovf file with the vSphere Client using the OVF Template Wizard.
